# Thai Gold API

API for checking latest Thai Gold prices

## Requirements

- Node LTS

## Installation

```sh
$ npm install
$ npm run build
$ npm run start
```

or

```sh
$ yarn
$ yarn build
$ yarn start
```

## API

This API is based on HTTPS requests and JSON responses. `https://gold-th.plutopon.site/`

### Get latest gold prices

##### API request

`GET /latest`
